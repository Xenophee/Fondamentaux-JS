Les sélecteurs

Exercice 1

Dans un fichier HTML, créer une balise h1 vide, ayant pour id 'text'.

Créer une variable "textToDisplay" javascript contenant le texte suivant : 'Comment devenir un AS en JS'.

A l'aide des sélecteurs javascript, ajouter dans la balise h1, le texte initialisé dans la variable "textToDisplay".

Exercice 2

Télécharger la ressource selecteurs.html.

Dans la console :

Afficher le contenu de la balise h1.

Afficher le contenu de la balise ayant pour class "description".

Afficher individuellement le contenu html de chaque balise "li".

Exercice 3

En utilisant la ressource selecteurs.html, modifier en javascript, le contenu des balises li de la manière suivante :
1 - Javascript
2 - PHP
3 - ...

Exercice 4

En utilisant la ressource selecteurs.html, afficher dans le champs caché la source de l'image.

Exercice 5

En utilisant la ressource selecteurs.html, ajouter une classe 'important' à la balise span, permettant de mettre en gras le texte.

Exercice 6

En utilisant la ressource selecteurs.html, ajouter une classe 'red' donnant la couleur rouge à l'item 'Javascript' dans la liste des langages de programmation.
